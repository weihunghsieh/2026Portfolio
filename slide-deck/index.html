I design enterprise-scale products with complex information architecture, such as
Microsoft Playwright Testing, a product for QA testers in organizations to test web applications.

Playwright is an open-source testing framework sponsored by Microsoft, which runs
only one configuration at a time (E.g. Chromium on MacOS) with simple reporting.

When we kicked off the project, there was no spec.
I was asked to “turn this into an enterprise product running on Azure”.

With the information architecture diagram as the source of truth, I started designing on every level from scratch.
